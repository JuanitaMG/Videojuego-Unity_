# The Elements Game

## Descripción

The Elements Game es un proyecto académico desarrollado en Unity como parte del proceso de formación en Ingeniería en Informática. El objetivo del proyecto fue aplicar conocimientos de programación orientada a objetos, lógica de programación y desarrollo de videojuegos mediante la creación de un entorno interactivo.

Durante el desarrollo se implementaron diferentes mecánicas de juego, programación de escenarios e interacción entre los elementos del videojuego utilizando C#.

## Tecnologías utilizadas

- Unity
- C#
- Visual Studio

## Estructura del proyecto

El repositorio contiene los archivos necesarios para abrir el proyecto en Unity:

- Assets
- Packages
- ProjectSettings

Las carpetas generadas automáticamente por Unity, como `Library`, `Temp` y `Logs`, no se incluyen en el repositorio.

## Cómo ejecutar el proyecto

1. Clonar o descargar este repositorio.
2. Abrir Unity Hub.
3. Seleccionar **Add Project** y elegir la carpeta del proyecto.
4. Esperar a que Unity genere los archivos necesarios.
5. Abrir la escena principal y ejecutar el proyecto desde el Editor de Unity.

## Objetivo académico

Este proyecto fue desarrollado con fines académicos para fortalecer competencias en:

- Desarrollo de videojuegos con Unity.
- Programación en C#.
- Diseño e implementación de mecánicas de juego.
